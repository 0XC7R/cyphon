############
Dependencies
############

Cyphon uses a number of third-party Python packages. The following packages are used by Cyphon Engine::

    beautifulsoup4 (bs4)
    bleach
    celery
    cryptography              # used by jira
    django
    django-autocomplete-light
    django-constance
    django-cors-headers       # CORS for development environment
    django-extensions         # used for model graphs
    django-filter
    django-grappelli
    django-mailbox
    django-picklefield        # used by django-constance
    djangorestframework
    djangorestframework-jwt
    docutils
    elasticsearch
    geoip2
    geopy
    httmock
    jira
    langdetect
    oauth2
    pika
    psycopg2
    pymongo
    python-dateutil
    pytz
    selenium
    six
    testfixtures
    tlslite
    tweepy
